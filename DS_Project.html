Auto Arabic Diacritization¶

The objective of this project was to develop a model capable of predicting Arabic diacritization for a provided input text. The project resources can be accessed through the following link: auto_arabic_diacritization.

The project is structured into three main components:

  1. Data Handling
  2. Model Creation and Training
  3. Result Discussion

1. Data Handling¶

The provided data consisted of raw Arabic text, for simplification purposes, I selected a single text file; however, the provided code is designed to accommodate more than on file. The input data for this process is undiacritized Arabic text, while the target class is the diacritization of the text. Preprocess steps:

  1. Splitting the data into sentences using symbols such as (\n, {, ], etc.), followed by the removal of symbols and links.
  2. Tokenize the data, diacritics, Arabic characters.
  3. Augmenting the data with Start-of-Sequence (SOS), End-of-Sequence (EOS), and Padding (PAD) tokens, as well as map the target class using one-hot encoding technique.

def fit(dataloader, num_epochs, char_size, class_siz, device, optimizer, criterion,lr = 1e-3):
  total_loss = []
  for epoch in range(num_epochs):
    model.train()
    running_loss = 0.0

    for batch in dataloader:
        inputs = batch[0]
        labels = batch[1]

        optimizer.zero_grad()
        outputs = model(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()

        running_loss += loss.item() * inputs.size(0)
    epoch_loss = running_loss / len(dataset)
    print(f'Epoch [{epoch+1}/{num_epochs}], Loss: {epoch_loss:.4f}')
    total_loss.append(epoch_loss)
  print('Training finished!')
  return total_loss

REV_CLASSES_MAPPING = {0:'<PAD>', 1: '<SOS>', 2:'<EOS>', 3:'', 4:'َ', 5 : 'ُ', 6: 'ِ', 7: 'ّ', 8: 'ْ', 9: 'ً', 11: 'ٌ', 12:'ٍ'}
def remove_diacritization( text):
    diacritic_pattern = re.compile(r'[\u064e\u064f\u0650\u0651\u0652\u064b\u064c\u064d]')
    cleaned_text = re.sub(diacritic_pattern, '', text)
    return cleaned_text

def predict(model, input_text, input_tokenizer,classes):
    # Tokenize the input text
    input_sequence = [input_tokenizer[char] for char in input_text]
    input_sequence = torch.tensor(input_sequence).unsqueeze(0)

    model.eval()
    decoder_input = torch.zeros((1, input_sequence.size(1)), dtype=torch.long)
    with torch.no_grad():
        input_sequence = input_sequence
        output_sequence = model(input_sequence)
        _, predicted_indices = torch.max(output_sequence, dim=2)

    output = ''
    # loop on input characters and predicted diacritizations
    for char, prediction in zip(remove_diacritization(input_text), predicted_indices.tolist()[0]):
        # append character
        output += char
        if char not in ARABIC_LETTERS_LIST:
            continue
        output += REV_CLASSES_MAPPING[prediction]

    return output
